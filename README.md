# 🚀 Servlet Login Application

A simple Java web application built using Servlets, JSP, and Apache Tomcat.

## 🔹 Features
- User Login System
- Session Management (Login & Logout)
- JSP + Servlet Integration
- Clean UI with CSS

## 🛠️ Tech Stack
- Java
- Servlets (Jakarta EE)
- JSP
- Apache Tomcat
- HTML, CSS

## ▶️ How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat server
3. Run project on server
4. Open:
   http://localhost:8080/ServletApp/
